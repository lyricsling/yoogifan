## Hugo 简单配置
- 使用命令安装 hugo
	```
	sudo pacman -S hugo
	```
- 创建网站

	使用命令创建网站，网站的名称为 my_blog。
	```
	hugo new site my_blog
	```
	命令执行成功后，就会在目录中生成 my_blog 文件夹。文件夹中包含了hugo框架所需文件。
